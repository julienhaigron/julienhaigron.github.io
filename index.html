Seance


Description:

Seance is an online multiplayer video game, in which the three players will incarnate themselves in the middle of a card game.
They will outsmart challenges from the game master. The true nature of the game is revealed when a powerful demon, who's haunting the game, starts possessing the game master.
Players are then allowed to cheat in order to face those obstacles.
Watch out not getting caught.. You may regret it quickly those actions.

My Contribution:

As the only game programmer for this project, my job was to:
- Define the architecure with the game designers
- Develop the core systems (client & server side)
- Create tools designed for Game Designers in order to add data quickly
